Adding Various Columns with Functions¶

In the previous section we created an advanced analysis with merge and groupby and other Pandas capabilities, which are harder to do in Excel. In this section, we will check the more traditional functions that are often used in Excel spreadsheet and we will see how to use them easily in Pandas.

Loading Data¶

As usual, let’s load a dataset to work on. We will take another dataset that is used in machine learning education, “Bike Share”. This is a data set about the demand of bike share service in Seoul. Please note that we need to modify the default encoding of read_csv from ‘UTF-8’ to ‘latin1’.

import pandas as pd

df = pd.read_csv('https://archive.ics.uci.edu/ml/machine-learning-databases/00560/SeoulBikeData.csv', encoding='latin1')

Date Columns¶

We will start with creating a few columns based on the date column, such as day of the week and month. First we will make sure that the column is date column according to the format that we see (note the day before the month), and then we can use it to calculate various built-in date functions. To use the built-in functions we need to access the date accessor (.dt)

(
    df
    .assign(Date = pd.to_datetime(df.Date, format='%d/%m/%Y', errors='coerce'))
    .assign(DOW = lambda x : x.Date.dt.dayofweek)
    .assign(month = lambda x : x.Date.dt.month)
)
